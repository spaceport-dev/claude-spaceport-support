# Spaceport Support — Claude Code Plugin Marketplace

A Claude Code plugin marketplace for the [Spaceport](https://spaceport.dev) framework.

## Installation

From your Spaceport project directory, in Claude Code:

```
/plugin marketplace add spaceport-dev/claude-spaceport-support
/plugin install spaceport@spaceport-support
```

After install, restart Claude Code to load the agents.

To update later:

```
/plugin marketplace update spaceport-support
```

## Plugins

| Plugin | Description |
|---|---|
| [`spaceport`](plugins/spaceport/) | Six specialized agents and full framework documentation reference for building Spaceport applications. |

See the individual plugin READMEs for details on what each one provides.
